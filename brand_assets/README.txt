HuyBuilds Studio — Brand / Logo Assets
======================================

VECTOR MASTERS (scalable, editable — use these wherever possible)
  logo-tile.svg ........ Primary monogram tile (terracotta, "hb" + amber dot, rounded, transparent corners)
  logo-tile-mono.svg ... 1-color version (ink tile, white letters) for stamps, faxes, etch
  wordmark.svg ......... Horizontal wordmark "HuyBuilds" + amber dot
  favicon.svg .......... Small-size mark — single "h" + dot (use below ~24px)

  Note: the SVGs reference the Google Font "Hanken Grotesk" via @import, so they render
  correctly in any browser. If you open them in a vector editor without the font installed,
  install Hanken Grotesk (Google Fonts, free) or convert text to outlines for a portable file.

RASTER / PNG (true Hanken Grotesk, ready to upload — full-bleed terracotta square)
  logo-app-icon-512.png .. App icon / social profile picture (Facebook, Zalo, etc.)
  logo-app-icon-256.png .. Medium raster
  logo-app-icon-180.png .. Apple touch icon
  favicon-64.png ......... Favicon (large)
  favicon-32.png ......... Favicon (standard browser tab)

  For a 16px favicon, use favicon.svg (the single-"h" mark) — "hb" is too tight at 16px.

COLORS
  Terracotta #C2603A · Amber dot #CC9A46 · Off-white letters #FAF5ED · Ink #2C2620

RULES
  Always keep the amber dot — it's the signature. Don't recolor outside the palette, add
  gradients, stretch, rotate, or change the typeface (always Hanken Grotesk 800).
